# Mini-Game-Hub

Mini Game Hub is a Java Swing-based GUI application that combines multiple mini-games into a single interactive platform. The project focuses on creating a simple, user-friendly environment where users can enjoy different logic-based games from one dashboard.
<br>
📌 Features
🧩 Multiple mini-games in one application
🖥️ Simple and interactive GUI using Java Swing
🎯 Logic-based and engaging gameplay
🔄 Smooth navigation between games
⏱️ Timer-based challenges for better user experience
🎲 Included Games
Word Jumble
Displays a shuffled word
User has to guess the correct word
Math Challenge
Random arithmetic questions
Includes a timer for added difficulty
Sudoku (4×4)
Small grid-based logic game
Validates user input based on rules
Guess the Letter
Hidden word guessing game
Letters are revealed as the user guesses correctly
🛠️ Technologies Used
Java
Java Swing (GUI)
Object-Oriented Programming (OOP)
Event Handling
